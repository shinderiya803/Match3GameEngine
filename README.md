# Match3GameEngine

A console-based Match-3 game inspired by Candy Crush, developed in C++ using Object-Oriented Programming.

## Features

* 8×8 game board
* Random candy generation
* Adjacent swapping
* Horizontal and vertical match detection
* Removal of matched candies
* Gravity effect
* Board refill
* Score system
* Move counter
* Target score
* High score file handling
* Colored console output

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling
* Standard Template Library (STL)
* Visual Studio

## Future Improvements

* Graphical interface using SFML
* Multiple levels
* Animations
* Shuffle when no moves are available
* Leaderboard system

## How to Run

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Build and run the project.
